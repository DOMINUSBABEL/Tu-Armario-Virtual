# DY(DressYourself) 👗✨

Welcome to **DY(DressYourself)**, a modern, highly interactive, and visually stunning cross-platform app inspired by the aesthetic and gamification dynamics of *Dress to Impress*.

## 🚀 Features
- **Avatars & Customization:** Build and style your avatar.
- **Virtual Wardrobe:** Explore, collect, and organize unique fashion items.
- **Runways & Challenges:** Participate in time-limited global fashion challenges to showcase your unique style.
- **Leaderboards & Badges:** Earn Style Points (SP) to climb global rankings and show off your badges.

## 🛠️ Tech Stack
- **Kotlin Multiplatform (KMP):** Share logic across Android, iOS, and Desktop.
- **Compose Multiplatform:** Write UI once, run everywhere.
- **Android SDK:** Fully built, tested, and deployed for Android via Gradle and GitHub Actions.
